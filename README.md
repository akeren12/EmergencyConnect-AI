# EmergencyConnect-AI

[![Django CI](https://github.com/akeren12/EmergencyConnect-AI/actions/workflows/django.yml/badge.svg)](https://github.com/akeren12/EmergencyConnect-AI/actions/workflows/django.yml)

EmergencyConnect AI is an AI-powered emergency assistance platform designed to help users during medical emergencies, accidents, and disaster situations.

## Problem

During emergencies, people often panic and may not know the right actions to take. Delays in communication and lack of access to reliable guidance can increase risks during critical situations.

## Planned Features

* One-tap SOS alert
* Emergency contact management
* AI emergency assistant
* Emergency severity assessment
* AI-generated emergency guidance
* Nearby hospital, blood bank, and pharmacy recommendations

## Tech Stack

* Python
* Django
* Django REST Framework
* PostgreSQL / Supabase
* OpenAI API
* Railway
